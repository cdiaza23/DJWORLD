# DJ WORLD 🎵

Un sitio web para DJ profesionales con herramientas de conversión de audio, gestión de paquetes y mucho más.

## 🌐 Ver en línea (GitHub Pages)

Tu sitio está disponible en:
**https://cdiaza23.github.io/DJWORLD/**

## 📁 Estructura del proyecto

```
DJWORLD/
├── index.html              # Página principal (Inicio)
├── INICIO/
│   ├── inicio.html        # Página de inicio alternativa
│   └── index.css          # Estilos de inicio
├── CONVERTIDOR/
│   ├── convertidor.html   # Convertidor de audio
│   └── convertidor.css    # Estilos convertidor
├── PAQUETES/
│   ├── paquetes.html      # Página de paquetes
│   ├── paquetes.css       # Estilos paquetes
│   ├── COMPRA/
│   │   ├── compra.html    # Catálogo de compra
│   │   └── compra.css     # Estilos compra
│   └── DESCARGA/
│       ├── descarga.html  # Paquetes para descargar
│       └── descarga.css   # Estilos descarga
├── img/                    # Imágenes del sitio
└── .nojekyll              # Archivo para GitHub Pages
```

## 🔗 Enlaces principales

- **Inicio**: index.html
- **Convertidor**: CONVERTIDOR/convertidor.html
- **Paquetes**: PAQUETES/paquetes.html
- **Compra**: PAQUETES/COMPRA/compra.html
- **Descarga**: PAQUETES/DESCARGA/descarga.html

## 🚀 Características

- ✅ Convertidor de video a audio
- ✅ Catálogo de herramientas DJ
- ✅ Paquetes de música por géneros
- ✅ Carrito de compras
- ✅ Diseño responsive
- ✅ Interfaz oscura con tema verde neón

## 📱 Optimizado para

- Computadoras
- Tablets
- Dispositivos móviles

## ⚙️ Configuración de GitHub Pages

El sitio está configurado para publicarse desde la rama `main` (o `master`) en la raíz del repositorio.

Para cambiar la rama o carpeta de publicación:
1. Ve a **Settings** del repositorio
2. Busca **Pages**
3. En "Source", selecciona tu rama y carpeta
4. Haz clic en Save

## 💡 Notas para desarrollo

- Todos los enlaces son relativos para funcionar correctamente en GitHub Pages
- Las imágenes están en la carpeta `img/`
- Los estilos CSS están en cada carpeta correspondiente
- El archivo `.nojekyll` evita que GitHub intente compilar con Jekyll
